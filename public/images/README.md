# Required Images

Your original HTML site referenced these image files, but they weren't
included in the files you uploaded (only the .html files were provided).
Add the following images to this folder (`public/images/`) using these
exact filenames, and everything will line up automatically:

| Filename               | Used for                                          |
|-------------------------|---------------------------------------------------|
| logo.png                | Site logo (header + browser tab favicon)           |
| backgroundImag.jpg       | Full-page fixed background (body)                  |
| backgroundm.jpg          | Services page header background                    |
| backgroundma.jpg         | Services section background                        |
| img1.jpeg                | Home "About" slider, image 1                       |
| img2.jpeg                | Home "About" slider, image 2                       |
| img3.jpeg                | Home "About" slider, image 3                       |
| 12.jpeg                  | Home "About" slider, image 4                       |
| 34.jpeg                  | Home "About" slider, image 5 / News card 1 image    |
| 78.jpeg                  | Home "About" slider, image 6 / News card 2 image    |
| 56.jpeg                  | Home "About" slider, image 7                       |

If you don't have one of the background images (`backgroundm.jpg` /
`backgroundma.jpg`), you can safely reuse `backgroundImag.jpg` for all
three — just duplicate the file under each name.

Any image not present will simply show as a broken image / missing
background in the browser; nothing else will break.
